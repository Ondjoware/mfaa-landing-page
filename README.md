# Landing Page Museu de Forças Armadas de Angola

Esta é a plataforma oficial de apresentação do Museu das Forças Armadas de Angola pela internet.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
